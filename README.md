## 🔧 **طرق التثبيت**

### 📲 **طريقة Termux (الأندرويد)**

```bash
# تحديث الحزم
pkg update && pkg upgrade -y

# تثبيت المتطلبات
pkg install git python -y

# تحميل الأداة
git clone https://github.com/IDLEBX/TBomb.git

# الدخول للمجلد
cd TBomb

# تثبيت المكتبات
pip install requests

# تشغيل الأداة
python bomber.py
```

💻 طريقة Linux (Kali/Ubuntu)

```bash
# تحديث النظام
sudo apt update && sudo apt upgrade -y

# تثبيت المتطلبات
sudo apt install git python3 python3-pip -y

# تحميل الأداة
git clone https://github.com/IDLEBX/TBomb.git

# الدخول للمجلد
cd TBomb

# تثبيت المكتبات
pip3 install requests

# تشغيل الأداة
python3 bomber.py
```
